# Show HN: RoleForge — 31 production-grade agent roles for AI teams

**Problem:** Building agent teams with CrewAI/LangGraph means writing ad-hoc system prompts. "You are a helpful assistant" doesn't produce a Credit Risk Analyst.

**Solution:** A framework-agnostic YAML library of 31 validated agent personas with ready-made adapters for CrewAI, LangChain, and LangGraph.

**What you get (free):**
- 3 starter roles (Data Scientist, Narrative Architect, Ethics Advisor)
- Full engine: adapters, registry, validators, graph templates
- 43 tests, JSON Schema validation
- One `pip install` away

**What's paid:** $49-$499 for domain-specific bundles (Audit, Risk, Governance, etc.)

**Why this matters:** We validated the market — no curated agent role library exists. Closest thing is prompt marketplaces selling $3 snippets. This is a professional tool, not a toy.

**Repo:** https://github.com/majidrajpar/roleforge

**Built with:** Python, Pydantic, YAML, and a lot of prompt engineering discipline.

**Tech note:** The core engine is MIT licensed. The role definitions are Elastic License 2.0 (free for personal/internal use, paid for commercial redistribution). We believe commercialization is a virtue — it funds maintenance and prevents abandonware.
